# XELIS VM - XVM

!!WORK IN PROGRESS!!

XELIS blockchain XVM syntax highlighting for VSCode.  
The file extension is `.xel`.

## Features

### Syntax highlighting

- Function
- Constant (numeric and boolean)
- Storage types
- Comment
- Builtin functions
- Keyword control

### IntelliSense

- Struct object and functions
- Builtin functions
